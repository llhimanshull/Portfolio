Full Stack

Beer Bazzar

Node.js , EJS , Express , HTML , JS , CSS , MongoDB

Beer Bazzar is an interactive web platform designed to provide extensive knowledge about beer. Built using MongoDB, the site efficiently manages data from both users and visitors. The website features a dedicated Admin Panel powered by Express.js, allowing authorized personnel to dynamically control the site's content and behavior. Leveraging Locomotive JS, EJS, and CSS, the platform ensures a smooth and engaging user experience. Visitors can actively participate by writing reviews, which are instantly displayed on the site, fostering a vibrant community. The platform also enables the admin or designated individuals to create and publish blog posts directly from the site's interface. For enhanced security and seamless authentication, the project integrates JWT tokens and cookies. Following the Model-View-Controller (MVC) architecture, the project maintains clean code organization and efficient data flow, ensuring scalability and performance.

Elit Vault

Java , Springboot , Postman , MongoDB

I developed Elit Vault, an exclusive invite-only luxury marketplace designed to provide a secure and seamless user experience. The platform leverages Spring Boot for backend development, with MongoDB as the database solution and Thymeleaf for dynamic content rendering. Following the MVC architecture, the project ensures modularity and efficient data flow with dedicated layers for beans, controllers, services, DAO, and database connections. A standout feature is the robust invitation code validation system, which grants access only to verified users. This feature includes dynamic invitation code management, where each code is marked as used upon successful registration, minimizing misuse and enhancing platform security.

To enhance usability, I implemented a detailed admin dashboard that provides comprehensive insights and control. Admins can efficiently create and manage invitations, track their usage, and oversee registered users. The project incorporates precise session management to ensure data privacy and secure access control. By emphasizing performance and scalability, the platform efficiently handles hundreds of invitations and thousands of user entries with optimized database queries.
